## Installing dependencies

Clone the project, open terminal, go to the project path ...

```bash
$ composer install
```

## Create the file .env

Copy the file .env.exemple and rename in .env

##  Migrate and seed database

```bash
$ php artisan migrate --seed
```

## Running the app

```bash
$ php artisan serve

```
